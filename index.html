Resources

Here is a list of resources I have used to teach myself HTML, CSS, JavaScript, PHP and SQL...

Dani Krossing

Daniel Nielsen is a Web Developer from Denmark who provides free web development courses through his YouTube channel.


Dani has been my primary source of videos for learning, with particular focus on his PHP course.


Coder Coder

Jessica Chan is from the United States and hosts a YouTube channel devoted to web development.


Hers is a mix of various tutorials and motiviational videos related to web development. 'How I learned to code and landed a job (no CS degree)' is a personal favourite of mine.


Kevin Powell

Kevin Powell is a CSS specialist! His YouTube channel is full of tutorials and even some CSS battles.


A brilliant resource for all things style based.


Aaron Jack

Aaron Jack is a former English teacher turned coder from the US who now uses his coding skills to live abroad.


His channel focuses more on the motivational elements and why coding IS FOR EVERONE!


Kalle Hallden

Kalle Hallden is a Swedish software developer who is self-taught.


His YouTube channel also focuses more on the motivational elements as well as various topics and challenges that he sets himself.


W3Schools

W3Schools is a repository of information for web development covering a range of topics from basic HTML & CSS all the way through to back-end development with SQL and PHP.


CSS Battle

An awesome website full of challenges to recreate various designs using CSS. Challenges range from beginner to expert level.


CSS Battle

Attempting to help coders to achieve mastery of their chosen language(s) through challenge.


JavaScript & JQuery by Jon Duckett

This book is to reference and understanding is what "console.log('test')" is to figuring out why my code isn't working...
